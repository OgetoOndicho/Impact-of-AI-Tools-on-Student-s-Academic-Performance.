# Impact of AI Tools on Student's Academic Performance.
## About the Project
This is an end to end Data Analytics project primarily utilizing MySQL to analyze data and quantify findings if use of AI tools has any effecect on Students performance at Okinawa University. The advancement of technology in Artificial Intelligence and Machine Learning tools has found its way into learning institutions around the world. Statistics estimate that approximately 54% of students use AI on a daily or weekly basis. Overally, approximately 86% of students use multiple AI tools worldwide according to https://www.demandsage.com/ai-in-education-statistics/ report.
This project, uses sample data from a hypothetical institution (Okinawa University) aims to investigate if indeed use of AI tools has any effect on students Grade Point Average. Using data metrics such as GPA Baseline and GPA Post AI scores, career confidence, majors and the numbers of weekly saved hours, this study seeks to discover student's performance post AI Tools use records any significant improvements when comparing with the baseline GPA scores (before AI Tools use).
The study also seeks to discover performance changes across the student population based on their respective average GPA Scores, and find out if use of AI affect students career confidence score.
